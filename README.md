# react-redux-universal-starter-kit
## About
A starter kit for a universal web app using react, redux
## Installation
```bash
npm install
```
## Running Dev Server

```bash
npm run dev
```

## Running Production Server

```bash
npm run build
npm run start
```

## Deployment on Heroku

```bash
heroku login
heroku create
git push heroku master
heroku open
```
